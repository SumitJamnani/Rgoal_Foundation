== Installation ==

1. Upload the entire `testimonial-builder` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Testimonial Builder Panels menu to customize this plugin.
4. Copy your publish Testimonial shortcode and then paste into your page or post to display it.

== Frequently Asked Questions ==

= Can I Add Multiple Testimonial with this plugin? =

Yes, you can add multiple testimonials with help of this plguin , you can change their position easily using plugin drag and drop api. 

= Can I able to display testimonial inside any post or page?  =

Yes, you can , with using testimonial shortcode  [TEST_B id=xx] (here xx your shortcode id) you can display your testimonial content on any page or posts.

= How to customize testimonial labels  font size, color and  other settings? =

You can customize every options from testimonial setting area. just go to testimonial plugin menu page add or edit  testimonial and then from sidebar setting area you can able customize your options.

= Can i able to add my testimonial profile images?  =  

Yes, you can able to upload your testimonial images.
 
= How to add testimonial in widget area ?  =  

Just open your widget menu then add "Text widget" , now paste your testimonial shortcode in this text widget.
