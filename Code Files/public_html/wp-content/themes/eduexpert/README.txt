== Description ==

EduExpert is the perfect for create an education website for any learning business related to education, teaching and learning and set up your site in a few steps. This WordPress Theme can be customized to your specific needs with incredible ease, thanks to drag and drop PageLayer page builder( with alot of components ). It has the customizing options like blog style, logo update, layout control, color modification and much more.
